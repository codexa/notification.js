notification.js

Web Notifications just couldn't be easier!



How notification.js works

Many websites and apps need to alert the user to changes in a minimalistic, yet attention-grabbing way. The Notifications API that was introduced in Firefox 4 provides the ideal solution to this issue. Unfortunately, the API never became standardized, and because of that, is only supported in Firefox.

Enter notification.js

notification.js allows websites to harness the power of notifications by just including a single script and stylesheet in their code. After that is done, the website can simply call the notification.show() function, and notification.js will either delegate the notification to the Notifications API (if the user is using Firefox 4+), or it will create a notification on top of the actual website.


How to use notification.js

  1. Download notification.js from here.
  2. Extract the zip, and put the files in a folder on your website.
  3. In the <head> section of your site's code, paste this:

    <!-- notification.js -->
    <script type="text/javascript" src="http://site.org/folder-containing-files/notification.js"></script>
    <link rel="stylesheet" type="text/css" href="http://site.org/folder-containing-files/notification.css" />

  4. Correct the urls to point to the notification.js files.
